# Number Guessing Game ( Pragramm changed with new )

A simple **Node.js console application** where the user guesses a number until they get it right.

## Features

- Keeps asking the user for input until the correct number is guessed
- Supports quitting the game with 'q'
- Easy to run in Node.js
